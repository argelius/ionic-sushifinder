ionic-sushifinder
=================

Sushi search app

Running
-------

Install ionic command line tool

```bash
sudo npm install -g ionic
```

Install dependencies

```bash
npm install
```

Install plugins

```bash
ionic plugin add org.apache.cordova.geolocation
```

Add platforms, e.g.

```bash
ionic platform android
ionic platform ios
```

Deploy to device

```bash
ionic run android
```
